# Fizyka---karta-wzorow
Opracowanie edukacyjne inspirowane zakresem kart wzorów do egzaminu maturalnego publikowanych przez Centralna Komisja Egzaminacyjna. Układ, forma i sposób prezentacji treści zostały opracowane samodzielnie. Materiał nie stanowi oficjalnego dokumentu CKE i ma charakter pomocniczy.
